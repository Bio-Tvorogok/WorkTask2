TestXml - файл для теста
Предполагается, что файл не содержит ошибок и пригоден для чтения программой.

Добавление, удаление и редактирование элементов происходит через выскакивающее меню, на правую клавишу мыши
При работе с ячейкой предприятия происходит проверка на совпадения, если схожее название найдено, то оно добавлено не будет.
При работе с ячейкой сотрудника происходит проверка на ввод числа в нужное поле, иначе ячейка добавлена не будет.
В Xml файлах данной программы отсутствует поля для суммарного дохода. Данное значение высчитывается по ходу обработки Xml файла или добавления(изменения) ячеек.
